Librería de actividades E-learning

The following library is developed in jQuery and you can follow it through gitHub or download it through of this page. This project constitutes a joint work with Andrés Quiroga (LinkedIn) , an excellent Software Developer.

You can adapt the code and appearance to your design based on the HTML blocks together with a CSS style file (activities.css) and another that includes the name of each of the activities for example: crossword.js, mtples_answers.js, multiple_choice.js, drag_drop.js depending on the activity you want to include.

These are the following activities developed so far:

  1. Complete texts (multiple_choice.js)
  2. Select answer (select.js)
  3. Drag and drop (drag_drop.js)
  4. Multiple options with single answer (mtple_unique_answer.js)
  5. Multiple options with multiple answers (mtples_answers.js)
  6. False and true (false_true.js)
  7. Drag and drop of images (drag_drop_images.js)
  8. Selection from accordion (accordion_select.js)
  9. Crossword (crossword_puzzle.js)
  10. Extensive content evaluation (eval_big_content.js)
  11. Concentrate (concentrate.js)
