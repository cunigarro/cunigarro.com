Librería de actividades E-learning

La siguiente librería esta desarrollada en jQuery y podrás seguirla a través de gitHub o descargarla a través de ésta página. Este proyecto constituye un trabajo conjunto con Andrés Quiroga (LinkedIn), un excelente Desarrollador de Software.

Puedes adaptar el código y la apariencia a tu diseño basado en los bloques de HTML junto con un archivo de estilos CSS (activities.css) y otro que incluye el nombre de cada una de las actividades por ejemplo: crossword.js, mtples_answers.js, multiple_choice.js, drag_drop.js dependiendo de la actividad que desés incluír.

Estas son las siguientes actividades desarrolladas hasta el momento:

  1. Completar textos (multiple_choice.js)
  2. Seleccionar respuesta (select.js)
  3. Drag and drop (drag_drop.js)
  4. Multiples opciones con única respuesta (mtple_unique_answer.js)
  5. Multiples opciones con múltiples respuestas (mtples_answers.js)
  6. Falso y verdadero (false_true.js)
  7. Drag and drop de imágenes (drag_drop_images.js)
  8. Selección a partir de acordeón (accordion_select.js)
  9. Crucigrama (crossword_puzzle.js)
  10. Evaluación de contenidos extensos (eval_big_content.js)
  11. Concéntrese (concentrate.js)
